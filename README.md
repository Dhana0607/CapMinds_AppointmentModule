# 📅 CapMinds Appointment Scheduler

A simple and responsive appointment scheduling web app with a calendar interface and dashboard.

## 🚀 Features
- Monthly calendar view
- Book, edit, and delete appointments
- Dashboard with search and filters
- Local storage support
- Responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ Run Locally
Just open `index.html` in your browser.
